2.2.3.2. Edit Non-Login User Account

Upon successful account initialization, the SimpelELN MultiTeam Edition is automatically configured with a Super User account. Subsequently, the holder of this Super User account is authorized to create and manage additional Super User, Team Admin and Team User accounts as required. Upon successful login, users with administrative privileges are authorized to modify the accounts of other users. Specifically, the Super User account can manage both Team Admin and Team User accounts, whereas Team Admin and Lab Manager accounts are restricted to managing only the Team User accounts within their respective teams.

2.2.3.2.1. Edit Registered User Account

  1. Edit Registered User Account by Super User

    Upon successful login, users with administrative privileges are authorized to modify the accounts of other users. Specifically, the Super User account can manage both Team Admin and Team User accounts. For a detailed and stepwise guide on editing a registered user’s profile by a Super User account, kindly refer to the guidelines provided below.

    • Log in as Super User.

      • Navigate to the URL http://<host_ip>:<port>/login using a web browser, making sure to replace <host_ip> with the IP address of your server host machine and <port> with the designated port number.

      • For example, replace the <port> with the server.port number specified in the application.properties file, the default is 32780.

    • Navigate to the User List page.

      • Option 1: From the Admin page (http://<host_ip>:<port>/admin/elnadmin), select Users User List in sequence to redirect to the User List page.

      • Option 2: From the Edit User page (http://<host_ip>:<port>/editloginuser), select System Admin Users User List in sequence to redirect to the User List page.

    • Under the Users tab, click on the Edit link to redirect to a page designed for editing user profiles.

    • Complete and submit the User Details form.

    • Please refer to the reference Edit Registered User Account by Super User for detailed step-by-step instructions on how to edit a registered user’s account.

  2. Edit Registered User Account by Team Admin

    Upon successful login, users with administrative privileges are authorized to modify the accounts of other users. Specifically, Team Admin and Lab Manager accounts can manage the Team User accounts within their respective teams. For a detailed and stepwise guide on editing a registered user’s profile, kindly refer to the guidelines provided below.

    • Log in as either Team Admin or Lab Manager.

      • Navigate to the URL http://<host_ip>:<port>/login using a web browser, making sure to replace <host_ip> with the IP address of your server host machine and <port> with the designated port number.

      • For example, replace the <port> with the server.port number specified in the application.properties file, the default is 32780.

    • Navigate to the User List page.

      • Option 1: From the Projects page (http://<host_ip>:<port>/project), select User Admin Group Users User List in sequence to redirect to the User List page.

      • Option 2: From the User Admin page (http://<host_ip>:<port>/useradmin), select Group Users User List in sequence to redirect to the User List page.

    • Under the Users tab, click on the Edit link to redirect to a page designed for editing user profiles.

    • Complete and submit the User Details form.

    • Please refer to the reference Edit Registered User Account by Team Admin for detailed step-by-step instructions on how to edit a registered user’s account.
